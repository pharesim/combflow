1. Operator

HiveComb is a content discovery and publishing interface for the Hive blockchain (hereinafter "we", "us", or "HiveComb").

2. Description of Service

HiveComb is a content discovery and publishing interface for the Hive blockchain. The service:

  • Indexes and classifies publicly available Hive blockchain posts by topic, language, and sentiment.
  • Provides a browseable interface for discovering content.
  • Allows authenticated users to create posts and comments that are broadcast to the Hive blockchain via the Hive Keychain browser extension.
  • Allows authenticated users to report misclassified posts.

HiveComb is a read-and-relay interface — we do not host or control the Hive blockchain. Content published through HiveComb is stored on the decentralised Hive blockchain, not on our servers.

6. Automated Classification

HiveComb uses machine learning to classify posts. This classification:

  • Is organisational, not editorial — it groups content by topic for discovery.
  • May be imperfect — we provide a misclassification report feature for corrections.
  • Affects visibility in browse results (unclassified posts don't appear in the main feed but remain accessible via direct links).
  • Does not constitute a judgement on content quality, truthfulness, or value.
